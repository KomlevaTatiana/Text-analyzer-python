# Text-analyzer-python
Beginner Python project for text analysis and visualization 

# Features
- user can type or paste text
- user can load a '.txt' file
- counts:
  - total characters with spaces
  - total characters without spaces
  - total words
  - total sentences
  - total paragraphs
  - total unique words
- shows the 10 most common words
- creates a bar chart with summary statistics

# Tools used
- Python
- 're'
- 'collections.Counter'
- 'matplotlib'
- Google Colab

# File
- 'text_analyzer.ipynb'  main notebook with the project code
- 'sample_text.txt'  example text file for testing

# Purpose
I created this project to practice text processing, counting word frequencies, and visualizing text statistics in Python.

# How to run
Open the notebook in Google Colab or Jupyter Notebook and run the cells.
